The Outlook prerequisites:


A Microsoft Entra ID tenant. If you don't have a tenant, [create an Azure account](https://azure.microsoft.com/pricing/purchase-options/azure-account/).

Once you've created the app registration, you can get the **Client ID** and **Tenant** values from the app registration's **Overview** section in the Azure portal.

Copy and store the secret that you create by using the app registration's **Certificates & secrets > Client secrets > New client secret** link. The secret will appear in the **Client Credential** field.
