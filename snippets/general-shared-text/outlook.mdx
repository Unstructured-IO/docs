The Outlook prerequisites:


A Microsoft Entra ID tenant. If you don't have a tenant, [create an Azure account](https://azure.microsoft.com/en-us/pricing/purchase-options/azure-account/).

Once you've created the application you can access data for **Client ID** and **Tenant** fields in App registrations' **Overview** section.

Copy and store the secret created in App registrations' **Certificates & secrets > Client secrets > New client secret.** for the **Client Credential** field.
