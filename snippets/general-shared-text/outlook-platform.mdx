Fill in the following fields:

- **Name** (_required_): A unique name for this connector.
- **User Email** (_required_): Outlook email to download messages from.
- **Folders** (_required_): Comma separated list of email folders to download messages form. Do not provide subfolders! In order to access messages from a subfolder, refer to **Recursive** field.
- **Client ID** (_required_): Azure AD App client ID.
- **Client Credential** (_required_): Azure AD App client secret.
- **Authentication Provider URL**: Authentication token provider for Microsoft apps. By default https://login.microsoftonline.com is used.
- **Tenant**: ID or domain name associated with your Azure AD instance. By defaults "common" is used.
- **Recursive**: Check this box to recursively access messages from subfolders.
