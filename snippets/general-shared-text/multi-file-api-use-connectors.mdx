<Note>The `partition` method in the Python and JavaScript/TypeScript SDKs process single files only. To process multiple files at a time, use the [Unstructured Ingest CLI](/ingestion/overview#unstructured-ingest-cli) or [Unstructured Ingest Python](/ingestion/overview#unstructured-ingest-python) with their provided [source connectors](/api-reference/ingest/source-connectors/overview) and [destination connectors](/api-reference/ingest/destination-connector/overview).</Note>
