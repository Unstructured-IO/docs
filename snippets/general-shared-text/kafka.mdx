The Kafka prerequisites:

bootstrap_server: The hostname of the bootstrap Kafka cluster to connect to.  
port: The port number of the cluster, typically `9092`.
topic: The unique name of the category to read messages from and write messgages to on the cluster. 
confluent: True to indicate that the cluster is running Confluent Kafka.
num_messages_to_consume: The maximum number of messages to get from the topic. The default is `1`.
timeout: The maximum amount of time to wait for the response of a request to the topic, expressed in seconds. The default is `1.0`.
