Fill in the following fields:

- **Name** (_required_): A unique name for this connector.
- **Remote URL** (_required_): The full SFTP server URL (for example, **sftp://host:port/path/**).
- **SFTP username**: The username for logging in to the SFTP server.
- **Password** (_required_): The password associated with the username.
- **Uncompress**: Check this box if there are compressed files that require decompression.
- **Recursive**: Check this box to recursively access files from subdirectories.
