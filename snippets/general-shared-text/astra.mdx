The Astra DB connector dependencies:

```bash CLI, Python
pip install "unstructured[astra]"
```

These environment variables:

- `ASTRA_DB_ENDPOINT` - The API endpoint for your Astra DB database. To get the endpoint, see the **Database Details > API Endpoint** value on your database's **Overview** tab.
- `ASTRA_DB_TOKEN` - The database application token for your target Astra DB database.
- `ASTRA_DB_KEYSPACE` - The name of the namespace in your Astra DB database.
- `ASTRA_DB_COLLECTION` - The name of the collection in your namespace. 
- `ASTRA_DB_EMBEDDING_DIMENSIONS` - The number of embedding dimensions in the processed data.
