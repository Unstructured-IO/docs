Fill in the following fields:

- **Name** (_required_): A unique name for this connector.
- **Bucket Name** (_required_): The name of the Amazon S3 bucket.
- **AWS Access Key**: The AWS access key ID, if the bucket is private.
- **AWS Secret Key**: The AWS secret access key corresponding to the preceding AWS access key ID.
- **Token**: If required, the security token for temporary access.
- **Endpoint URL**: A custom URL, if connecting to a non-AWS S3 bucket.
- **Recursive**: Check this box to ingest data from subfolders within the bucket.
- **Connect Anonymously** (_source connectors only_): Check this box if connecting to a bucket with public access, and you do not want to associate the connection with your account.
