- `<name>` (_required_) - A unique name for this connector.
- `hosts` is a _required_ array, with `<host-url>` specifying the URL of the Elastic Cloud instance to access, for example: `["https://<random-number>.<region>.<cloud-provider>.cloud.es.io"]`.
- `<es-api-key>` (_required_) - The Elastic Cloud API key for the target cluster.
- `<index-name>` (_required_) - The name of the target index in the cluster.
