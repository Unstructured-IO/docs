The Azure AI Search (formerly Azure Cognitive Search) prerequisites:

- The API key for Azure AI Search.
- The endpoint for Azure AI Search.
- The name of the index in Azure AI Search.

See the [Azure AI Search documentation](https://docs.microsoft.com/azure/search/).

<Warning>
    Make sure that you use a compatible index schema. For reference, see the [sample index schema](/open-source/ingest/destination-connectors/azure-cognitive-search#sample-index-schema).
</Warning>
