This quickstart uses the Unstructured Python SDK to call the Unstructured API workflow operations to get your data RAG-ready. The Python code for this 
quickstart is in a remote hosted Google Colab notebook. Data is processed on Unstructured-hosted compute resources.

To run this quickstart, open the [notebook](https://colab.research.google.com/github/Unstructured-IO/notebooks/blob/main/notebooks/Dropbox_To_Pinecone_Connector_Quickstart.ipynb) and begin following the notebook's on-screen instructions.
