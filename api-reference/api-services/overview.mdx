The Unstructured Serverless API documentation covers the following API services:

* [Unstructured Serverless API](/api-reference/api-services/saas-api-development-guide): Serverless API hosted by Unstructured, scalable and secure, your data remains private, pay-as-you-go.
* [Free Unstructured API](/api-reference/api-services/free-api): Hosted by Unstructured, data processing is capped to 1000 pages, documents submitted to the free API may be utilized for our proprietary model training and evaluation purposes.
* [Unstructured API on Azure](/api-reference/api-services/azure) and [Unstructured API on AWS](/api-reference/api-services/aws): The functionality and security of the Unstructured Serverless API, but deployed through the Azure Marketplace or AWS Marketplace, in your preferred infrastructure.

## Supported File Types

Unstructured API supports the same file types as the open source Unstructured library.
However, the Unstructured API services include more powerful file transformation capabilities,
such as a more accurate table extraction model.

|Category  |File Types                                                    |
|----------|--------------------------------------------------------------|
|Plaintext |.eml, .html, .md, .msg, .rst, .rtf, .txt, .xml                |
|Images    |.png, .jpg, .jpeg, .tiff, .bmp, .heic                         |
|Documents |.csv, .doc, .docx, .epub, .odt, .pdf, .ppt, .pptx, .tsv, .xlsx|


## Data Ingestion

Unstructured API services support ingesting data from various sources - refer to [unstructured-ingest documentation](/open-source/ingest/overview) to learn more.
