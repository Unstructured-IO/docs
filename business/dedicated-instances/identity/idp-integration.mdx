## IdP Integration

### Overview

Dedicated Instances support SAML- and OIDC-based identity providers.

### Typical Steps

1. Configure your identity provider
2. Exchange metadata or connection details
3. Map roles for RBAC
4. Validate the login flow

See [Security](../security) for more information about access controls.
