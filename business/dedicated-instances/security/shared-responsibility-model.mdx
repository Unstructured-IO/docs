## What the shared responsibility model means
A shared responsibility model defines how security and operational responsibilities are divided between Unstructured and the customer.

Unstructured is responsible for the infrastructure and software it operates. Customers are responsible for their accounts, data, identities, access policies, and the network resources they control.

Major cloud providers use similar shared responsibility models for their own services.

The first table covers traffic to the Unstructured platform, including UI and API access from your environment. The second table covers traffic to your data sources, when Unstructured connects to resources in your cloud environment.
